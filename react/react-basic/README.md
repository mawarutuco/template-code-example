
# 筆記

記得安裝 nodejs~

這篇是 react 18

## 做了什麼指令

```bash

# 初始化 | npx是下載後自動砍掉不要的安裝包
npx create-react-app react-basic-app
# 去安裝好的資料夾
cd react-basic-app
# 啟動
npm start

# 完成開發
npm run build

```
